Projet de portfolio en analyse des données RH, portant sur des indicateurs clés RH tels que le suivi des performances et le taux d’attrition.
Le projet a consisté à nettoyer, transformer et visualiser les données afin de créer un tableau de bord.
J’ai utilisé Microsoft Excel et l’Éditeur de requêtes de Power BI pour le nettoyage et le prétraitement des données, ainsi que Power BI pour la création du tableau de bord, dans le cadre du parcours Data Analyst (Power BI) de Datacamp.
Remarque :
Le jeu de données initial contenait plus de 80 000 enregistrements, qui ont dû être nettoyés et préparés pour ce projet. Seul le jeu de données nettoyé, composé de 76 000 enregistrements, est présent dans le dépôt.
Outil de préparation des données : Microsoft Excel et Microsoft Power BI
Outil de visualisation : Microsoft Power BI
